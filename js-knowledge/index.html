What is Javascript?

  • JavaScript is a high-level, interpreted programming language that is widely used to create interactive and dynamic web content
  • It was first introduced in 1995
  • One of 3 main language on web programming: HTML, CSS, JavaScript
  • JavaScript is not Java

JS Just-In-Time

  1. JavaScript is an interpreted language which means that it does not need to be compiled before it can be executed.
  2. Two methods for converting source code into a form that can be run on a platform. Static compilation converts the code into a language for a specific platform. An interpreter directly executes the source code.
    JIT compilation attempts to use the benefits of both.
    3. Example: 
    
    console.log("Hello world")
  3. Tools like Babel that can transpile JavaScript code from newer versions of the language to older versions that are more widely supported by browsers and environments.

Data types

JS automatic assign data for a variable when assign a value. (Dynamic type - the data type of a variable can change during runtime)

  1. Number - used for numeric values.
  2. String - used for text values.
  3. Boolean - used for logical values (true/false).
  4. Undefined - used for variables that have not been assigned a value.
  5. Null - used to represent a deliberate non-value.
  6. Object - used for more complex data structures, such as arrays and functions.

Additionally, starting from ECMAScript 2015, JavaScript has two new data types:

  1. BigInt - used for integers with arbitrary precision.
  2. Symbol - used to create unique identifiers for objects.

Note:

typeof([variable]) return data type: number, string, boolean, object(null), function, undefined

Function Basic

  • Function is a collection of instruments
  • Syntax: 
    
    function nameOfFunction(var1, var2, var3, ...) { // var1, var2, var3 are agruments
      // Some code
      [return value;]
    }
  • 'return' is used to response data and exit function
  • Usage
    • 
    [const return value =] nameOfFunction(value1, value2, value3, …)

    Example:

    
    // example with: Given a positive integer, is it a prime number?
    function isPrimeNumber(number) {
      let isPrime = true
      if (number <= 1 ) {
          isPrime = false
      } else {
          for (let i = 2; i <= Math.sqrt(number); i++ ) {
              if (number % i == 0) {
                  isPrime = false;
                  break;
              }
          }
      }
      return isPrime
    }

Array methods

Array Method Description
push() Adds one or more elements to the end of an array and returns the new length of the array.
pop() Removes the last element from an array and returns that element.
shift() Removes the first element from an array and returns that element.
unshift() Adds one or more elements to the beginning of an array and returns the new length of the array.
indexOf() Searches an array for a specified element and returns its position (index) in the array. Returns -1 if the element is not found.
lastIndexOf() Searches an array for a specified element starting from the end of the array and returns its position (index) in the array. Returns -1 if the element is not found.
reverse() Reverses the order of the elements in an array.
slice() Extracts a section of an array and returns a new array.
splice() Adds or removes elements from an array and returns the removed elements.
join() Joins all elements of an array into a string. The elements are separated by a specified separator.
toString() Converts an array to a string and returns the result.
find() Returns the value of the first element in an array that satisfies the provided testing function.
filter() Creates a new array with all elements that pass a test (provided as a function).
map() Creates a new array with the results of calling a provided function on every element in the calling array.
reduce() Applies a function against an accumulator and each element in the array to reduce it to a single value.
concat() Merges two or more arrays, and returns a new array that contains all the elements of the original arrays.
sort() Sorts the elements of an array in place and returns the sorted array.
every() Checks if all the elements in an array pass a test (provided as a function). Returns true if all elements pass the test, false otherwise.

DOM - Document Object Model

About

The HTML DOM is a standard object model and programming interface for HTML.

Helping on operator of a website: change color, size, ..., noti error, event handler: onclick, hover, ...

The HTML DOM model is constructed as a tree of Objects


    // open browser => console tag
    document.body.style.backgroundColor = "red";

Function

To done it, DOM defines:

  • The HTML elements as objects
  • The properties of all HTML elements
  • The methods to access all HTML elements
  • The events for all HTML elements

How to use

  • Finding HTML elements
    • 
    document.getElementById(id);                  // Find an element by element id
    document.getElementByTagName(name);           // Find elements by tag Name
    document.getElementByClassName(name);         // Find elements by class name
    document.querySelector("[css-selector]");     // Find a element like css Selector
    document.querySelectorAll("[css-selector]");  // Find all elements like css Selector
  • Changing properties HTML elements
    • 
    // property
    element.innerHTML = new html content; // Change the inner HTML of an element
    element.attribute = new value; // Change the attribute of an HTML element
    element.style.property = new style ; // Change the style of an HTML element
    // note: property follow camel case. example: background-color => backgroundColor
    // method
    element.setAttribute(attribute, value); // Change the attribute of an HTML element
  • Add and Remove elements
    • 
    // document.createElement("element tag name"); create an HTML element
    // Ex:
    const divElem = document.createElement("div");
    const pElem = document.createElement("p"); // pElem.innerText = "Hello"
    
    // element.appendChild(element);  Add an HTML element to document
    // Ex:
    divElem.appendChild(pElem);

    // document.removeChild(element); remove an HTML element
    // Ex:
    divElem.removeChild(pElem);

    // document.replaceChild(new, old); Replace an HTML element
    const newPElem = document.createElement(p)
    divElem.replace(newPElem, pElem)
    
    // document.write(text); Write into the HTML output steam
    document.write("Hello")
  • Events Handlers
    • 
    // html
    <button id="js-btn-event-handler" onclick="alert('inline click')">
      Event Handler
    </button>

    // javascript
    const element = document.getElementById("js-btn-event-handler")

    element.addEventListener('click', function () {
      alert('add click');
    });
    element.addEventListener('click', function () {
      alert('add click 2');
    });

        if (posY + vecY > screenWidth - ballWidth || posY + vecY < 0) {
          nextY = randomVec(5);
          vecY *= -1;
          vecY = (vecY / Math.abs(vecY)) * nextY;
        }
    
        posX = posX + vecX; // posX += vecX
        posY = posY + vecY; // posY += vecY
    
        ballElem.style.transform = `translate(${posX}px,${posY}px)`;
        await sleep(10);
      }
    }
    
    function randomVec(maxValue) {
      return Math.floor(Math.random() * (maxValue + 1)) + 1;
    }
    
    async function sleep(ms) {
      return new Promise((resolve) => {
        setTimeout(resolve, ms);
      });
    }

    // player.js
    let screenWidth = 500;
    let screenHeight = 500;
    class Player {
      constructor(ctx, img, x, y, width, height) {
        this.ctx = ctx;
        this.img = img;
        this.x = x ? x : 0;
        this.y = y ? y : 0;
        this.width = width ? width : 100;
        this.height = height ? height : 100;
        this.step = 10;
      }

      draw() {
        this.ctx.drawImage(this.img, this.x, this.y, this.width, this.height);
      }

      moveRight = () => {
        if (this.x + this.step <= screenWidth - this.width) {
          this.x += this.step;
        }
      };

      moveLeft = () => {
        if (this.x - this.step >= 0) {
          this.x -= this.step;
        }
      };

      moveUp = () => {
        this.y -= this.step;
      };

      moveDown = () => {
        this.y += this.step;
      };
    }

    export default Player;
  

    // index.js

      setInterval(() => {
        // fps (frame per second!)
        fillBackground();
        player.draw();
