What is Javascript?

JavaScript is a high-level, interpreted programming language that is widely used to create interactive and dynamic web content. It was first introduced in 1995 and has since become an essential component of web development. JavaScript allows developers to add interactive features to websites, such as animations, pop-ups, and form validation, among other things. It is also commonly used in creating web applications, server-side programming, and mobile app development. JavaScript code can be embedded directly into HTML documents or included in separate files and linked to HTML documents. It can be executed on the client-side, using a web browser, or on the server-side, using a web server or a backend framework. JavaScript is often abbreviated as JS and is distinct from Java, despite their similar-sounding names.

JS Just-In-Time

  1. JavaScript is an interpreted languageJavaScript is an interpreted language, which means that it does not need to be compiled before it can be executed. Instead, JavaScript code is executed directly by a JavaScript engine in a web browser or a server-side runtime environment like Node.js
  2. Traditionally there are two methods for converting source code into a form that can be run on a platform. Static compilation converts the code into a language for a specific platform. An interpreter directly executes the source code.
    JIT compilation attempts to use the benefits of both.
    3. Example: 
    
    console.log("Hello world")
  3. Additionally, there are tools like Babel that can transpile JavaScript code from newer versions of the language to older versions that are more widely supported by browsers and environments. These tools typically use a combination of parsing, transformation, and code generation techniques to modify the code without fully compiling it

Using nodeJS

  • Writing simple code:
    •

    •

  • Create file: index.html and Embed JS file
    • 
  /* file: index.html */

  • Run index.html with Live Server and check out Console tab on Inspect browser

Data types

JS automatic assign data for a variable when assign a value. (Dynamic type - the data type of a variable can change during runtime)

  1. Number - used for numeric values.
  2. String - used for text values.
  3. Boolean - used for logical values (true/false).
  4. Undefined - used for variables that have not been assigned a value.
  5. Null - used to represent a deliberate non-value.
  6. Object - used for more complex data structures, such as arrays and functions.
  7. Symbol - used to create unique identifiers for objects.

Additionally, starting from ECMAScript 2015, JavaScript has two new data types:

  1. BigInt - used for integers with arbitrary precision.
  2. Symbol - used to create unique identifiers for objects.

Variables declaration

Keywords

Keywords for declare a variable var, let, and const 

                
    var data = {};
    let width = 100;
    const PI = 3.14;
  1. var: function scope or global scope and can be re-declared
  2. let: block scope and cannot be re-declared
  3. const: block scope and cannot be updated or re-declared

Variable naming rules:

  1. Variable name can not the same with keyword: var, for, if, ...
  2. Start with letter or undercode(_)
  3. No contain special characters, ex: &, *, (, ),...

Noted: Variable names are case sensitive, mean Name and name are different
