What is CSS?

CSS stands for Cascading Style Sheets. It is a style sheet language used to describe the presentation of a document written in HTML or XML (including XML dialects like SVG or XHTML). CSS is used to define styles for web pages, including layout, colors, fonts, and other visual elements.

Imbed CSS to HTML: 3 ways

  1. Inline style: apply CSS to individual HTML elements by using the "style" attribute 
    2. 
    <h1 style="color: red; font-size: 24px;">Hello World!</h1>
  2. Internal style sheet: add CSS to an HTML file by using the "style" tag within the head section of the HTML document 
    3. 
    <head>
      <style>
        h1 {
          color: red;
          font-size: 24px;
        }
      </style>
    </head>
    <body>
      <h1>Hello World!</h1>
    </body>
  3. External style sheet: link an external CSS file to your HTML file by using the "link" tag within the head section of the HTML document. 
    4. 
    <head>
      <link rel="stylesheet" type="text/css" href="style.css">
    </head>
    <body>
      <h1>Hello World!</h1>
    </body>
            
    
    /* style.css file */
    h1 {
      color: red;
      font-size: 24px;
    }

Syntax: 3 elements

  • Selector: HTML element (or elements) that the rule applies to
  • Property: the specific style property
  • Value: set for the property

    selector {
      property: value; /* example: color, margin, border */
      property: value; /* example:  red     10px   1px solid black */
      /* more properties and values... */
    }

Selectors

CSS provides a variety of selectors that allow you to target specific HTML elements on a webpage and apply styles to them. Here are some of the most common types of selectors in CSS:

Type of selectors

  1. Element selectors: These target specific HTML elements on the page, such as "p" for paragraphs or "h1" for headings. 
                
    /* Universal selectors (*) */: 
    * {margin: 0; padding: 0}
    p {
        /* styles applied to all paragraphs */
    }
    h1 {
        /* styles applied to all level 1 headings */
    }
  1. Class selectors: These target elements with a specific class attribute, using a period (.) followed by the class name. 
                
    .my-class {
      /* styles applied to all elements with class="my-class" */
    }
  1. ID selectors: These target elements with a specific ID attribute, using a pound sign (#) followed by the ID name. 
                
    #my-id {
      /* styles applied to the element with id="my-id" */
    }
  1. Attribute selectors: These target elements with a specific attribute value, using brackets [] and the attribute name and value. 
                
    a[href="https://example.com"] {
      /* styles applied to all links with href="https://example.com" */
    }

Combine Selectors

  1. Descendant selectors: These target elements that are descendants of another element, using a space between the selectors. 
                
    ul li {
      /* styles applied to all li elements that are descendants of a ul */
    }
  1. Child selectors: These target elements that are direct children of another element, using a greater than sign (>). 
                
    ul > li {
      /* styles applied to all li elements that are direct children of a ul */
    }
  1. Adjacent sibling selectors: These target elements that come immediately after another element, using a plus sign (+). 
                
    h1 + p {
      /* styles applied to the first p element that comes immediately after an h1 */
    }
  1. General sibling selectors: These target elements that come after another element, using a tilde (~). 
                
    h1 ~ p {
      /* styles applied to all p elements that come after an h1 */
    }
  1. More than one selector have same style: using a tilde (,). 
                
    h1, p {
      /* styles all p and h elements to red */
      color: red;
    }

Pseudo-selectors

  1. :hover - is used to select an element when the mouse pointer is over it.

  2. :active - is used to select an element when it is being clicked.

  3. :focus - is used to select an element when it is focused.

  4. :first-child - is used to select the first child element of a parent element.

  5. :last-child - is used to select the last child element of a parent element.

  6. :nth-child(n) - is used to select the nth child element of a parent element.

  7. :nth-of-type(n) - is used to select the nth element of a particular type within a parent element.

  8. :not(selector) - is used to select all elements that do not match the specified selector.

pseudo elements

  1. ::before - used to insert content before the content of an element
  2. ::after - used to insert content after the content of an element
  3. ::first-line - used to apply styles to the first line of text within an element
  4. ::first-letter - used to apply styles to the first letter of text within an element
  5. ::selection - used to apply styles to text that has been selected by the user

Units of measurement

The pixel (px): the standard unit of measurement

Others

  1. Percentages (%): Used to specify relative measurements relative to the size of the parent element.

  2. em: Used to specify font sizes relative to the font-size of the parent element.

  3. rem: Similar to em, but the font-size is relative to the root element instead of the parent element.

  4. vw and vh: Used to specify sizes relative to the viewport width and height, respectively.

  5. cm, mm, in, pt, and pc: Used to specify physical units of measurement like centimeters, millimeters, inches, points, and picas.

Inheritance

Refers to the way styles are passed down from a parent element to its child elements. 

            
    <style>
      body {
        font-size: 16px; /* this property will be inherited by all child elements */
      }
      .parent {
        color: blue;
      }
      .child {
        color: inherit; /* this will inherit the color from the parent element */
      }
    </style>
    
    <body>
      <h1>Welcome to my website!</h1>
      <div class="parent">
        <p>This text will be blue.</p>
        <p class="child">This text will also be blue.</p>
      </div>
    </body>

Inherited properties: :active, color, background, cursor, font, :focus, :hover, text-align, visited,...

Font

CSS provides a range of properties that can be used to control the font of text, including font-family, font-size, font-style, font-weight, line-height, text-align, and text-decoration. 

            
    
    body {
      /* Set the font family to Arial and the font size to 14px */
      font-family: Arial, sans-serif;
      font-size: 14px;
      /* Set the font style to italic */
      font-style: italic;
      /* Set the font weight to bold */
      font-weight: bold;
    }

    /* Set the line height to 1.5 times the font size */
    p {
      font-size: 16px;
      line-height: 1.5;
    }

    /* Set the text alignment to center */
    h1 {
      text-align: center;
    }

    /* Add an underline to a link */
    a {
      text-decoration: underline;
    }

Position: relative, absolute, fixed and static

the position property is used to specify how an element should be positioned on a web page

Value Description
static The default position value. Elements with position: static are positioned according to the normal flow of the document.
relative Allows an element to be positioned relative to its normal position. The element's position can be adjusted using the top, right, bottom, and left properties.
absolute Allows an element to be positioned relative to its nearest positioned ancestor (an ancestor with a position value other than static), or to the initial containing block if there is no such ancestor. The element's position can be adjusted using the top, right, bottom, and left properties.
fixed Positions an element relative to the viewport (the browser window), even if the page is scrolled. The element's position can be adjusted using the top, right, bottom, and left properties.

Example of relative


    .my-element {
      position: relative;
      top: 20px;
      left: 30px;
    }

Property Name Possible Values Description

Media Responsive

Media queries are used to create responsive designs in CSS

Media queries use the @media rule in CSS to specify a set of styles to apply when certain conditions are met, such as a minimum or maximum screen width. Here's an example of how to write a media query that applies to screens with a maximum width of 600 pixels: 


    @media only screen and (max-width: 600px) {
      /* CSS styles go here */
    }

    @media only all and (min-width: 320px) and (max-width: 374px){/* code */}

    @media only screen and (min-width: 374px) and (max-width: 424px){/* code */}

    @media only screen and (min-width: 425px) and (max-width: 767px){/* code */}

    @media (min-width: 1024px) and (max-width: 1439px){/* code */}

2D Transform

Modify the appearance and position of HTML elements by transforming their size, position, and orientation. Here are some common 2D transforms in CSS

  1. translate() - moves the element along the x and y-axis.
    2.             
    transform: translate(x-value, y-value);
  2. scale() - changes the size of the element.
    3.               
    transform: scale(x-value, y-value);
  3. rotate() - rotates the element clockwise around its center.
    4.               
    transform: rotate(angle);
  4. skew() - skews the element along the x and y-axis.
    5.               
    transform: skew(x-angle, y-angle);
  5. matrix() - combines multiple transforms into one.
    6.               
    transform: matrix(scaleX, skewX, skewY, scaleY, translateX, translateY);
Exmple 
            
    .my-element {
      transform: translate(50px, 50px) rotate(45deg) scale(2, 2);
    }

Transition

CSS transitions allow you to smoothly change the property values of an element over a specified duration. With transitions, you can create animations that make your website more interactive and engaging. 

            
    .my-element {
      /* property(bgc, width, heigt) , duration(1s),
        timing-function(ease, ease-in, ease-out, ease-in-out, linear) */
      transition: background-color 1s ease-in-out; 
    }
    /* or */
    .my-element {
      transition: background-color 1s ease-in-out, color 2s ease-out;
    }

To trigger the transition, you need to change the property value. For example, if you want to change the background color of the element, you can do it like this 

          
    .my-element:hover {
      background-color: red;
    }
    /* or */
    .my-element:focus {
      width: 100px;
    }

Animation

CSS animations allow you to add motion and transitions to your web page elements. Here is a basic example of how to create an animation in CSS: 

            
    /* Define the animation */
    @keyframes example {
      from {background-color: red;}
      to {background-color: yellow;}
    }
    
    /* Apply the animation to an element: transition the background color of an element from red to yellow in 3s*/ 
    div {
      animation-name: example;
      animation-duration: 3s;
    }
    /* full syntax: animation : [name] [duration] [timing] [delay] [interaction-count] [direction][play-state] */
Properties
Property Description Values
Name The name of the animation Naming, example: slide-in
Duration The length of time the animation lasts 1s
Timing Function The rate of change of the animation over time ease-in-out
Delay The amount of time to wait before starting the animation 0.5s
Interaction Count The number of times the animation repeats infinite
Direction The direction of the animation on each iteration alternate
